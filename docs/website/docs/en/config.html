Settings & Privacy

Config file is YAML: default path ./config/config.yaml (auto-generated on first startup). Refer to template ./config/config.yaml.example.

You Probably Only Need to Change These

diff:
  enabled: true
  watch_paths:
    - "C:\\Users\\Dev\\Projects\\Go-Backend"

browser:
  enabled: true
  history_path: "" # empty for auto-detect

storage:
  db_path: "./data/workmirror.db"

privacy:
  enabled: true

AI Key & Environment Variables

By default, ai.provider=default uses the author's OpenAI-compatible gateway (default model llama-3.3-70b, endpoint https://gpt.devbin.de/proxy/mirror). To fully customize or use your own keys, switch to openai/anthropic/google/zhipu.

Recommended to inject Key via environment variables (avoid writing to disk/committing to repo), e.g., OPENAI_API_KEY, ANTHROPIC_API_KEY, GOOGLE_API_KEY, ZHIPU_API_KEY, SILICONFLOW_API_KEY.
