隐私策略

最后更新：2025年12月

复盘镜遵循 Local-first & Privacy by default：默认本地存储、默认脱敏、默认可离线。

数据存储

采集数据（窗口事件、Diff、浏览历史等）会写入本地 SQLite 数据库（默认 ./data/workmirror.db）。本地 HTTP 仅监听 127.0.0.1，不会暴露到局域网/公网。

默认脱敏

默认开启脱敏：URL 会清空 query/fragment 并保留域名与"存在路径"的事实；窗口标题/浏览标题会按规则替换敏感片段。你可以在配置中调整 privacy.patterns。

AI 处理

不配置任何 Key 时，不会调用外部 AI。启用 AI 后，仅在需要生成语义摘要/建议时发送经脱敏的最小必要文本到你配置的提供商（例如 DeepSeek / SiliconFlow）。
