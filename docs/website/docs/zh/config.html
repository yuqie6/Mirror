设置与隐私

配置文件为 YAML：默认路径 ./config/config.yaml（首次启动会自动生成）。可以参考模板 ./config/config.yaml.example。

你大概率只需要改这几项

diff:
  enabled: true
  watch_paths:
    - "C:\\Users\\Dev\\Projects\\Go-Backend"

browser:
  enabled: true
  history_path: "" # 为空则自动探测

storage:
  db_path: "./data/workmirror.db"

privacy:
  enabled: true

AI Key 与环境变量

建议通过环境变量注入 Key（避免写入磁盘/提交仓库），例如 DEEPSEEK_API_KEY、SILICONFLOW_API_KEY。
